# wangzhi
功能：保存平时重要的网址，
可以自动根据网址获得网页的小标签（成功率不高）和title（基本上都能成功）。

实现：自己获取网页小标签和title都是通过Java网页爬虫，将需要的信息爬取下来，
但是不知道为什么小标签的爬取成功率就是不高。
